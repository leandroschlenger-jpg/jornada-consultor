📋 Como usar o robô Python

  1. Na aba Projetos, clique em ▶ Gerenciar na linha de cada cliente
  2. Informe quantos meses para trás e o navegador, depois clique em Aplicar Períodos
  3. Clique em Exportar JSON ou Copiar Comando para passar a config ao robô Python
  4. O robô executa esocial_download_navegador_logado.py e solicita os downloads no portal
  5. O script de consulta verifica se os arquivos estão prontos e atualiza o status via API abaixo
  6. Status Disponível ✓ = arquivo pronto para baixar (status final)
API DO ROBÔ — BROWSER CONSOLE / PYTHON REQUESTS
// Atualizar um período específico
window.jdcRobo.esocial.setPeriodo('pid', 'MM/YYYY', 'disponivel');

// Atualizar vários de uma vez
window.jdcRobo.esocial.setLote([
  {pid: 'cliente1', periodo: '01/2024', status: 'disponivel'},
  {pid: 'cliente2', periodo: '02/2024', status: 'processando'}
]);

// Listar todos com status pendente/solicitado
window.jdcRobo.esocial.getAllParaDownload();
